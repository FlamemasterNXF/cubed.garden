cubed.garden

cubed.garden is a Minecraft SMP with proximity voice chat and regular events.

cubed.garden runs on minecraft 1.20.1 and requires the simple voice chat mod.

this is a step-by-step guide on how to install simple voice chat using fabric as if you have never installed minecraft mods before.
steps 0 and 1 can be safely skipped if you already have fabric installed.

step 0: download fabric

step 1: download the fabric api
step 2: download simple voice chat

step 3: navigate to C:/Users/[username]/AppData/Roaming/.minecraft
hint: in order to reach /AppData/Roaming type "%appdata%" in the search menu

step 4: create a folder called 'mods' inside /.minecraft.

step 5: paste the fabric api and simple voice chat files into /.minecraft/mods




congratulations, you have installed the simple voice chat mod (and the fabric mod loader)! press capslock to talk and v for more settings.

refer to the mod page for further information on the mod if you're interested.
